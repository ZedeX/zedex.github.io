English Name: WordPress Plugin: Access Keys Simplified Chinese Version?

### 插件名称 Plugin Name

快捷键（Access Keys）

### 功能描述 Description

本插件允许你给分类(Category)和页面(Page)增加快捷键，方便终极用户访问。插件很小，速度很快，几乎不占用系统资源。  
This plugin allows you to add Access Keys to Category and Page navigation menus to make your website far more accessible.

### 支持版本 Supported Version

WordPress 2.1+ (2.5可用)

### 用法 Usage

1、上传到WordPress插件目录；  
2、在后台插件管理页面中激活该插件；  
3、到后台在“管理”中找到本插件(Access Keys)；  
4、阅读其指南，并继续下一步操作。

### 补充信息

不同的浏览器设定了不同的快捷键，下面的列表提供了常用浏览器的快捷键组合方法：

- Firefox 2.0+：同时按住Alt键和Shift键，按快捷键；
- IE 5+：按住Alt键，按快捷键，同时松开两个键后按回车(Enter)；
- IE 4, Firefox 1.0, Mozilla或Netscape 7+：按住Alt键，按快捷键；
- Mac OS X下，使用Firefox, Mozilla, Netscape 7+, Internet Explorer 5+, Safari或Omniweb浏览器：按住Ctrl键，按快捷键；
- 使用Konqueror浏览器：按一下Ctrl键，再按快捷键；
- Mac下，使用IE 4.5和所有平台下Netscape 6以前的版本：不支持快捷键。
