English Name: WordPress Plugin: Akismet Simplified Chinese Version?

### 插件名称 Plugin Name

Akismet

### 功能描述 Description

Akismet通过全球服务器搜集所有的垃圾信息来自动判断你文章中的评论是不是垃圾评论（广告spam）。  
Akismet is a spam-fighting service that it checks the content of the comment anonymously with an online server, to determine whether it is spam or not.

### 支持版本 Supported Version

WordPress 任何现存的版本

### 用法 Usage

1、上传到WordPress插件目录；  
2、在后台插件管理页面中激活该插件；  
3、到后台在“设置”中找到本插件(Akismet)；  
4、输入WordPress API Key；  
5、如果你不知道该怎么做，请参照页面上的说明。

### 补充信息

Akismet使用全球统一的数据库来过滤垃圾评论，任何一个WordPress博客提交的垃圾评论都会被Akismet吸收核实并通知全球其他的WordPress博客，这样一旦其他博客收到类似的评论，Akismet将自动截获它们并在它们被显示到网站访问者之前需通过管理员的审核。

通过类似杀毒软件的病毒提交过程或者Wiki百科全书一样的功能，达到一人编辑，人人受用的效果。

这是一个环保插件。
